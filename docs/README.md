# Docs Index

`docs/` 目录现在按用途拆成 5 类，避免开发文档、联调文档和面试材料混在一起。

## 目录划分

### `architecture/`

- 系统设计与结构说明
- 适合看“项目为什么这样设计”

包含：

- `KnowFlow-V1-Design.md`
- `KnowFlow-V1-Playground-Design.md`
- `KnowFlow-V1-ByteCoach-Comparative-Analysis.md`

### `plans/`

- 开发计划、阶段实施计划、分任务执行文档
- 适合看“项目是怎么分阶段推进的”

包含：

- `KnowFlow-V1-Implementation-Plan.md`
- `KnowFlow-V1-Plan.md`
- `KnowFlow-V1-Playground-Implementation-Plan.md`
- `KnowFlow-V1-Polish-Implementation-Plan.md`
- `KnowFlow-V1-Knowledge-Writeback-Closure-Implementation-Plan.md`
- `knowflow-v1-tasks/`

### `testing/`

- Postman、curl、验证清单
- 适合联调、回归验证和接口测试

包含：

- `KnowFlow-V1-Postman-Guide.md`
- `KnowFlow-V1-Curl-Guide.md`
- `KnowFlow-V1-Verification-Checklist.md`
- `postman/`

### `demo/`

- 演示路径、联调操作手册
- 适合项目展示或跟别人一起走查

包含：

- `KnowFlow-V1-Demo-Guide.md`

### `career/`

- 简历、项目亮点、面试讲法
- 适合准备项目介绍和面试表达

包含：

- `KnowFlow-V1-Resume-Notes.md`
- `KnowFlow-V1-Interview-Guide.md`
